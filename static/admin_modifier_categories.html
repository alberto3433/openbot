About Modifier Categories

Modifier Categories define groups of add-ons that customers can ask about, like "what sweeteners do you have?" or "what toppings are available?"

Static Categories have a fixed description (e.g., sweeteners list). Dynamic Categories load options from the Ingredients table.

Aliases are keywords that trigger this category. For example, "sweetener, sugar, sugars" all map to the "sweeteners" category.
